## 1. 准备工作

### 1.1 准备发布文件

确保你已经有以下文件：

| 文件 | 说明 |
|:---|:---|
| `UPX_GUI.exe` | 编译后的主程序 |
| `UPX_GUI_Info.txt` | 软件说明文档 |
| `UPX_CMD_Shell.bat` | 命令行工具 |
| `version.txt` | 版本检测文件 |
| `upx-*.exe` | 各版本 UPX 工具 |
| 源代码文件 | 可选，用于开源 |

### 1.2 版本号规范

使用语义化版本控制：`v主版本号.次版本号.修订号`

| 示例 | 说明 |
|:---|:---|
| `v1.0.0` | 第一个正式版本 |
| `v1.0.1` | 修复 bug |
| `v1.1.0` | 添加新功能 |
| `v2.0.0` | 重大更新 |

## 2. 创建 GitHub 仓库

### 2.1 登录 GitHub
访问 [https://github.com](https://github.com)，使用账号登录。

### 2.2 创建新仓库

1. 点击右上角 **+** → **New repository**
2. 填写信息：

| 字段 | 填写内容 |
|:---|:---|
| **Repository name** | `upx-gui`（或你喜欢的名称） |
| **Description** | `UPX GUI - 强大的压缩工具，支持防误报优化` |
| **Visibility** | ✅ **Public** |
| **Initialize with** | ✅ **Add a README file** |
| | ✅ **Add .gitignore** → 选择 `C++` |
| | ✅ **Choose a license** → 选择 `MIT License` |

3. 点击 **Create repository**

## 3. 上传源代码

### 3.1 方法一：网页上传

1. 进入仓库页面，点击 **Add file** → **Upload files**
2. 拖拽源代码文件夹到上传区域
3. 填写 **Commit message**：`Initial commit - Source code`
4. 点击 **Commit changes**

```bash

git config --global user.email "你的邮箱"

# 克隆仓库
git clone https://github.com/你的用户名/upx-gui.git
cd upx-gui

# 复制源代码
# 将你的源代码文件复制到这里

# 提交代码
git add .
git commit -m "Initial commit"
git push origin main
```

## 4. 创建 version.txt（用于自动更新）

### 4.1 创建文件

1. 在仓库中点击 **Add file** → **Create new file**
2. 文件名：`version.txt`
3. 内容：

```
version 2.0.0

【更新日志】
1. 新增在线升级检测功能
2. 优化防误报压缩参数
3. 支持自定义压缩率设置
4. 修复已知问题
```

4. 点击 **Commit new file**

### 4.2 获取 Raw 链接

1. 点击 `version.txt` 文件
2. 点击右上角 **Raw** 按钮
3. 复制地址栏中的 URL：

```
https://raw.githubusercontent.com/你的用户名/upx-gui/main/version.txt
```

## 5. 构建便携式版本

### 5.1 运行部署脚本

在项目目录运行以下命令（或运行 `deploy_complete.bat`）：

```powershell
$qtBin = "D:\Computer\Qt653\6.5.3\mingw_64\bin"
$qtPlugins = "D:\Computer\Qt653\6.5.3\mingw_64\plugins"
$projectDir = "c:\Users\Journeyer\Desktop\upx2026"
$deployDir = "$projectDir\deploy_complete"

# 创建目录
New-Item -ItemType Directory -Force -Path "$deployDir\platforms" | Out-Null
New-Item -ItemType Directory -Force -Path "$deployDir\styles" | Out-Null

# 复制主程序
Copy-Item "$projectDir\UPX_GUI.exe" "$deployDir\"
Copy-Item "$projectDir\upx-*.exe" "$deployDir\"
Copy-Item "$projectDir\UPX_GUI_Info.txt" "$deployDir\"
Copy-Item "$projectDir\version.txt" "$deployDir\"
Copy-Item "$projectDir\UPX_CMD_Shell.bat" "$deployDir\"

# 复制 Qt DLL
Copy-Item "$qtBin\Qt6Core.dll" "$deployDir\"
Copy-Item "$qtBin\Qt6Gui.dll" "$deployDir\"
Copy-Item "$qtBin\Qt6Network.dll" "$deployDir\"
Copy-Item "$qtBin\Qt6Widgets.dll" "$deployDir\"
Copy-Item "$qtBin\libgcc_s_seh-1.dll" "$deployDir\"
Copy-Item "$qtBin\libstdc++-6.dll" "$deployDir\"
Copy-Item "$qtBin\libwinpthread-1.dll" "$deployDir\"

# 复制插件
Copy-Item "$qtPlugins\platforms\qwindows.dll" "$deployDir\platforms\"
Copy-Item "$qtPlugins\styles\qwindowsvistastyle.dll" "$deployDir\styles\"

# 打包成 ZIP
Compress-Archive -Path "$deployDir\*" -DestinationPath "$projectDir\UPX_GUI_v2.0.0_Portable.zip" -Force
```

### 5.2 验证构建结果

部署完成后，`deploy_complete` 文件夹应包含：

```

├── UPX_GUI.exe
├── UPX_GUI_Info.txt
├── UPX_CMD_Shell.bat
├── version.txt
├── upx-1.25-win32.exe
├── upx-2.3-win32.exe
├── upx-4.2.2-win32.exe
├── upx-4.2.2-win64.exe
├── upx-5.1.1-win32.exe
├── upx-5.1.1-win64.exe
├── upx-5.2.0-win32.exe
├── upx-5.2.0-win64.exe
├── Qt6Core.dll
├── Qt6Gui.dll
├── Qt6Network.dll
├── Qt6Widgets.dll
├── libgcc_s_seh-1.dll
├── libstdc++-6.dll
├── libwinpthread-1.dll
├── platforms/
│   └── qwindows.dll
└── styles/
    └── qwindowsvistastyle.dll
```

## 6. 创建 GitHub Release

### 6.1 进入 Releases 页面

1. 在仓库页面右侧边栏找到 **Releases**
2. 点击 **Create a new release**

### 6.2 填写发布信息

#### Section 1: Tag version
- 点击 **"Tag: Select tag"**
- 输入 `v2.0.0`
- 选择 **"Create new tag: v2.0.0"**

#### Section 2: Release title
- 输入：`Version 2.0.0`

#### Section 3: Release notes
输入更新日志：

```markdown
## What's New

### ✨ Features
- 🔄 **Auto Update**: 新增在线升级检测功能
- 🎯 **Anti-Virus Optimization**: 优化防误报压缩参数
- ⚙️ **Custom Compression**: 支持自定义压缩率设置
- 📥 **Drag & Drop**: 支持文件拖放功能

### 🐛 Bug Fixes
- Fixed: 文件菜单无效问题
- Fixed: 拖放功能失效问题  
- Fixed: UPX版本自动检测问题

### 📦 Downloads
- UPX_GUI_v2.0.0_Portable.zip (Windows Portable)
```

#### Section 4: Attach binaries
- 向下滚动找到 **"Attach binaries"** 区域
- 点击上传 `UPX_GUI_v2.0.0_Portable.zip`

#### Section 5: Publish
- ✅ **Set as the latest release**
- 点击 **"Publish release"**

## 7. 配置自动更新

### 7.1 修改程序配置

在 `mainwindow.h` 中修改：

```cpp
#define APP_VERSION "2.0.0"
#define UPDATE_URL "https://raw.githubusercontent.com/你的用户名/upx-gui/main/version.txt"
#define DOWNLOAD_URL "https://github.com/你的用户名/upx-gui/releases/latest"
```

### 7.2 重新编译

修改配置后重新编译程序，确保新版本支持自动更新检测。

## 8. 发布后维护

### 8.1 更新版本

当发布新版本时：

1. **修改 GitHub 上的 `version.txt`**：
   - 更新版本号：`version 2.0.1`
   - 更新日志内容

2. **创建新 Release**：
   - Tag: `v2.0.1`
   - 上传新的 `UPX_GUI_v2.0.1_Portable.zip`
   - 填写更新说明

3. **更新程序配置**（可选）：
   - 更新 `mainwindow.h` 中的版本号
   - 重新编译
